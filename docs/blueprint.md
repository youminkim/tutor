# **App Name**: ExamAce

## Core Features:

- Image Capture: Capture exam problem via mobile camera.
- Problem Analysis: Use OCR to extract text from the image and provide educational advice using a tool to mimic an experienced PSLE tutor. Provide clear explanations and identify key concepts.
- Concept Storage: Save a summary of the key concepts from the problem to local storage for later revision.
- Revision History: Allow users to review previously analyzed problems and summaries in a 'History' tab.

## Style Guidelines:

- Primary color: HSL 48, 95%, 50% - Vibrant gold (#FDD207) evokes a sense of achievement and academic excellence.
- Background color: HSL 48, 20%, 95% - Very light off-white (#FDFDF0), provides a clean and focused backdrop that minimizes distraction and maximizes readability.
- Accent color: HSL 18, 75%, 40% - Dark orange (#B84B0F) highlights important information and calls to action with a clear visual cue, adding a touch of energy and emphasis.
- Clean, legible font optimized for mobile screens to ensure comfortable reading.
- Simple, intuitive icons to represent different subjects and functions.
- Mobile-first design with a clear, single-column layout for easy navigation.
